# MD-student

## env create

use uv to contral product env
[uv documentation](https://docs.astral.sh/uv/)

### install (macos)

```bash
curl -LsSf https://astral.sh/uv/install.sh | bash
```

### use sync to create .venv

```bash
uv sync
```

## config

- change setting.py
- create .env file & add api_key

## Run

use uv env run

```bash
python3 main.py
```

## Build

```bash
pythin3 build.py
```
